 Strapi Local Setup & Content Creation

1 — Create Strapi Project
Used the official CLI to create a project:
npx create-strapi-app@latest strapi-assignment

2 — Run Strapi Locally
cd strapi-assignment
npm run develop

http://localhost:1337/admin

3 — Create Sample Content Type (Blog)

Used Content-Type Builder in admin panel.
